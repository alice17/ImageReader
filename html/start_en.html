Clarifai Account

You need to create a Clarifai account in order to use this extension.
Follow this steps:

  • Go to https://developer.clarifai.com/signup/ and sign up for a free API key. You need to verify your email address in order to use more than 100 monthly requests.
  • Go then to https://developer.clarifai.com/account/applications/ and click "My First Application".
  • In the section "Scopes" select the option "Predict on Public and Custom Models" and then click on "Create API key".
  • Once finished loading copy the API key to the clipboard.
  • Open ImageReader option page in the Chrome Extension tab and paste your API key.

OCR.Space Account (optional)

In order to use the text recognition inside images you need to create a OCR.Space account.

  • Go to http://space.us11.list-manage1.com/subscribe?u=ce17e59f5b68a2fd3542801fd&id=252aee70a1 and sign up for a free API key. You need to verify your email address in order to receive your API key via email.

  • Open ImageReader option page in the Chrome Extension tab and paste your API key.

Now you can already use ImageReader! But before, take a look at the options you can use:

  • Text recognition: if contained, extracts text in images.
  • Analyze private images: analizes no-public images.
  • Automatic page language recognition: inserts the description into the language used on the page.
  • Default labeling language: always inserts the description in a given language. If the automatic page recognition is enabled, it will be used this language if it's not detected any in the page.
  • Accuracy threshold: determines the percentage of accuracy of the description.
