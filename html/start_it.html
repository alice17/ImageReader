Benvenuto su ImageReader

Account di Clarifai

Devi creare un account Clarifai per usare questa estensione.
Segui questi passi:

  • Vai su https://developer.clarifai.com/signup/ e iscriviti per ottenere la API key gratuita. Per usufruire di più di 100 richieste mensili dovrai verificare la tua email.
  • Vai poi su https://developer.clarifai.com/account/applications/ e seleziona "My First Application".
  • Nella sezione "Scopes" seleziona l'opzione "Predict on Public and Custom Models" e premi il pulsante "Create API key".
  • Una volta finito il caricamento copia negli appunti la tua API key.
  • Apri la pagina delle opzioni di ImageReader nel tab delle estensioni di Google Chrome e incolla la tua API key.

OCR.Space Account (opzionale)

Per usare anche la funzione di text recognition all'interno delle immagini devi creare un account OCR.Space.

  • Vai su http://space.us11.list-manage1.com/subscribe?u=ce17e59f5b68a2fd3542801fd&id=252aee70a1 e iscriviti per ottenere la API key gratuita. Dovrai verificare la tua email per ricevere tramite posta elettronica la tua API key.
  • Apri la pagina delle opzioni di ImageReader nel tab delle estensioni di Google Chrome e incolla la tua API key.

Ora potrai usare ImageReader! Ma prima dai un'occhiata alle opzioni che puoi attivare:

  • Text recognition: estrae il testo dalle immagini se contenuto in esse.
  • Analizza immagini private: analizza immagini con URL privato.
  • Ricognizione automatica del linguaggio della pagina: la descrizione delle immagini è inserita nella lingua in cui è scritta la pagina web.
  • Linguaggio di default per il labeling: la descrizione è inserita sempre in una lingua a scelta dell'utente. Nel caso in cui sia attivata la ricognizione automatica della lingua della pagina, viene usata questa lingua nel caso in cui non venisse rilevata.
  • Soglia di accuratezza: determina la soglia di accuratezza della descrizione.
