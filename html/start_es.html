Bienvenido en ImageReader

Cuenta de Clarifai

Necesitas de crear un account de Clarifai para utilizár esta extensión.
Sigue estes pasos:

    • Ve á la página https://developer.clarifai.com/signup/ y regístrate para una API key gratis. Necesitas de verificár tu correo electronico para utilizár más de 100 pedidos mensuales.
  • Luego ve á la página https://developer.clarifai.com/account/applications/ y selecciona "My First Application".
  • En la sección "Scopes" selecctiona la opción Predict on Public and Custom Models" y luego selecciona "Create API key".
  • Una vez terminado de cargar copia la API key.
  • Abre la página de los opciónes de ImageReader en el tab de las extensiónes de Chrome y pega tu API key.

Cuenta de OCR.Space (opciónal)

Para utilizár el reconocimiento de texto dentro las imágenes tienes que crear una cuenta OCR.Space.

  • Ve a la página http://space.us11.list-manage1.com/subscribe?u=ce17e59f5b68a2fd3542801fd&id=252aee70a1 y regístrate para una API key gratis. Necesitas de verificar tu correo electronico para recibir tu API key.

  • Abre la página de las opciónes de ImageReader en el tab de las extensiónes de Chrome y pega tu API key.

Ahora ya puedes utilizár ImageReader! Pero antes, mira a las opciónes que puedes usar:

  • Reconocimiento de texto: si contenido, extrae el texto de las imágenes.
  • Analiza imágenes privadas: analiza imágenes con URL privado.
  • Reconocimiento automático de el idioma de la página: inserta la descripción en el idioma utilizádo en la página.
  • Idioma de labeling predeterminado: siempre inserta la descripción en un dado idioma. Si el reconocimiento automático de el idioma de la página esta habilitado, será utilizádo este idioma si no se detecta alguno en la página.
  • Umbral de precisión: determína la porcentage de precisión de la descripción.
Gracias para utilizár ImageReader.
